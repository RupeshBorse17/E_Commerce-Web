# E-Commerce Platform

**Full-Stack Project | Spring Boot, React.js, PostgreSQL, REST APIs**  
*Jan 2026 – Mar 2026*

## Project Overview
A **full-stack e-commerce application** that allows users to browse products, manage carts, and perform keyword-based searches.

## Features
- Product CRUD operations via **Spring Boot REST APIs**
- **React.js frontend** integrated with backend using Axios
- Image upload & retrieval using **MultipartFile**
- Database management with **PostgreSQL** using JPA/Hibernate
- Keyword-based product search and filtering

## Tech Stack
- **Frontend:** React.js, Axios, Tailwind CSS
- **Backend:** Spring Boot, Java, REST APIs
- **Database:** PostgreSQL
